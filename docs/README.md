# Shared notes

Use this folder for monorepo-level interview prep notes that are not owned by a
single project.

## Conventions

- Keep project-specific docs inside the project itself
- Put reusable notes, prompts, and study material here
- Prefer small focused files over one large running document

## Suggested files

- `system-design.md`
- `behavioral.md`
- `backend-topics.md`
- `questions-to-practice.md`

## Current shared material

- `typescript/` - TypeScript topics and data structure practice notes
